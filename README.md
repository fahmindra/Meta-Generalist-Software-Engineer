# Meta Generalist Software Engineer
The Meta Generalist Software Engineer Certificate is awarded to learners who complete : 
- both the Meta Front-end and Back-End Developer Certificates,
- as well as the Meta Database Engineer Certificate and
- one of the mobile development certificates, either iOS Developer or Android Developer certificate.

## Professional Certificate Course
1. Meta Front-End Developer
2. Meta Back-End Developer
3. Meta Database Engineer
4. Meta Android Developer
